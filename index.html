Monalisa

Basicamente, o projeto consiste na criação de um rosto com figuras geométricas como circulo, triangulo e até mesmo uma reta, para poder fazer o rosto nos programos, utilizamos os códigos para altura do olho e até mesmo comprimento, assim conseguimos desenvolver o projeto, além disso, definimos as cores do rosto.

O Javasccript foi utilizado para estilizar o rosto, empregando um esquema de cores, figuras geométricas e a altura, largura e comprimento

Os scripts do Javascript, consistem na aplicação de códigos para definir como as formas geométricas são representadas e manipuladas através de dados como coordenadas, tamanhos e ângulos. Elas podem ser definidas em código para serem desenhadas ou utilizadas em cálculos geométricos. Cada ponto de uma forma geométrica é definido por suas coordenadas no espaço (eixo X, eixo Y, e, em alguns casos, eixo Z). A dimensão de uma forma (largura, altura, raio, etc.) é armazenada como um valor numérico. Os ângulo são utilizados para representar a orientação e a inclinação das formas.
